# LeoRent — Платформа оренди квартир у Львові

LeoRent — це веб-платформа для пошуку та оренди квартир у Львові. Підтримує три ролі користувачів: покупець, власник та ріелтор. Інтегрований AI-агент дозволяє шукати житло у вільній формі, а також автоматично верифікує документи власників перед публікацією оголошення.

## Можливості

- **AI-пошук** — опишіть що шукаєте у вільній формі, AI підбере відповідні варіанти
- **Три ролі:** покупець, власник, ріелтор
- **Верифікація власників** — автоматична перевірка документів через AI перед публікацією оголошення
- **Каталог оголошень** з фільтрами (ціна, район, кількість кімнат тощо)
- **Автентифікація** з розмежуванням прав доступу по ролях

## Стек технологій

- React 18 + TypeScript
- Tailwind CSS v4
- React Router v7
- Vite 8

## Встановлення та запуск

```bash
git clone https://github.com/your-username/leorent-frontend.git
cd leorent-frontend
npm install
npm run dev
```

Frontend: `http://localhost:5173`
